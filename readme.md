Tech stack used -> Spring Boot, maven, RestTemplate and JUnit, Spring Boot Test
To see the solution using IntelliJ:
1) Run BPDTSApplication.java
2) You will need to get this message, [main] com.bpdts.rest.BPDTSApplication: Started BPDTSApplication in 1.222 seconds (JVM running for 1.729)
3) Then open browser and start typing the following URL's
  a) http://localhost:8080/user/{id}
  b) http://localhost:8080/users
  c) http://localhost:8080/city/{city}/users
4) Also a unit test class which can be executed to test the scenarios
